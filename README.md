# 🍽️ Omnifood Clone

> *Because cooking is hard... and cloning websites is easier.* 😄

A responsive clone of the **Omnifood Landing Page** built using **HTML, CSS, and a little bit of JavaScript**. This project was created to practice front-end development concepts like responsive layouts, animations, smooth scrolling, and writing cleaner HTML & CSS.

It isn't connected to any backend, so sadly you can't actually order food... you'll have to cook or order from somewhere else. 🍕

---

## 🚀 Why I Built This

I wanted to improve my front-end skills by recreating a professionally designed landing page instead of building something random from scratch.

This project helped me understand:

- How real landing pages are structured
- Responsive web design
- CSS positioning and layouts
- Hover effects and animations
- Smooth scrolling navigation
- Writing cleaner and reusable CSS

## 🏃 Running the Project

Nothing fancy here.

Just clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/rishitajain7/foodpage-demo
```
That's it. No npm. No frameworks. No "works on my machine" excuses.
